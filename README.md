# GINITA рЃърЃарЃЮрЃћрЃЦрЃбрЃў ­ЪќЦ№ИЈ

GINITA рЃљрЃарЃўрЃА рЃарЃћрЃљрЃџрЃБрЃа рЃЊрЃарЃЮрЃерЃў PC-рЃА рЃЏрЃЮрЃюрЃўрЃбрЃЮрЃарЃўрЃюрЃњрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃњрЃљрЃФрЃџрЃћрЃЋрЃЌ рЃЌрЃЋрЃљрЃџрЃДрЃБрЃарЃў рЃљрЃЊрЃћрЃЋрЃюрЃЮрЃЌ рЃЊрЃљ рЃЏрЃљрЃарЃЌрЃЮрЃЌ рЃЏрЃарЃљрЃЋрЃљрЃџрЃў рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃў рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃцрЃўрЃџрЃўрЃљрЃџрЃерЃў. рЃљрЃњрЃћрЃЉрЃБрЃџрЃўрЃљ Python Flask-рЃАрЃљ рЃЊрЃљ Socket.IO-рЃќрЃћ рЃарЃћрЃљрЃџрЃБрЃа рЃЊрЃарЃЮрЃерЃў рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.

## ­ЪїЪ рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃў

### рЃарЃћрЃљрЃџрЃБрЃа рЃЊрЃарЃЮрЃерЃў рЃЏрЃЮрЃюрЃўрЃбрЃЮрЃарЃўрЃюрЃњрЃў
- CPU-рЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ рЃЊрЃљ рЃгрЃљрЃарЃЏрЃљрЃЊрЃЮрЃЉрЃўрЃА рЃЏрЃћрЃбрЃарЃўрЃЎрЃћрЃЉрЃў
- RAM-рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃЌрЃЋрЃљрЃџрЃДрЃБрЃарЃўрЃА рЃЊрЃћрЃЋрЃюрЃћрЃЉрЃљ
- рЃЊрЃўрЃАрЃЎрЃўрЃА рЃљрЃЦрЃбрЃўрЃЋрЃЮрЃЉрЃљ рЃЊрЃљ рЃАрЃљрЃфрЃљрЃЋрЃў
- рЃЦрЃАрЃћрЃџрЃўрЃА рЃАрЃбрЃљрЃбрЃБрЃАрЃў рЃЊрЃљ рЃњрЃљрЃЏрЃбрЃљрЃарЃБрЃюрЃљрЃарЃўрЃљрЃюрЃЮрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ
- AnyDesk-рЃўрЃА рЃўрЃюрЃбрЃћрЃњрЃарЃљрЃфрЃўрЃљ рЃЊрЃўрЃАрЃбрЃљрЃюрЃфрЃўрЃБрЃарЃў рЃгрЃЋрЃЊрЃЮрЃЏрЃўрЃАрЃЌрЃЋрЃўрЃА
- рЃЎрЃарЃўрЃбрЃўрЃЎрЃБрЃџрЃў рЃарЃћрЃАрЃБрЃарЃАрЃћрЃЉрЃўрЃА рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃћрЃЉрЃў
- рЃарЃћрЃљрЃџрЃБрЃа рЃЊрЃарЃЮрЃерЃў рЃгрЃљрЃарЃЏрЃљрЃЊрЃЮрЃЉрЃўрЃА рЃњрЃарЃљрЃцрЃўрЃЎрЃћрЃЉрЃў

### рЃцрЃўрЃџрЃўрЃљрЃџрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ
- рЃЦрЃЋрЃћрЃЦрЃАрЃћрЃџрЃќрЃћ рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџрЃў рЃцрЃўрЃџрЃўрЃљрЃџрЃћрЃЉрЃўрЃА рЃЮрЃарЃњрЃљрЃюрЃўрЃќрЃћрЃЉрЃљ
- рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃћрЃЉрЃўрЃА рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљ
- рЃцрЃўрЃџрЃўрЃљрЃџрЃћрЃЉрЃА рЃерЃЮрЃарЃўрЃА рЃАрЃгрЃарЃљрЃцрЃў рЃюрЃљрЃЋрЃўрЃњрЃљрЃфрЃўрЃљ
- рЃарЃћрЃљрЃџрЃБрЃа рЃЊрЃарЃЮрЃерЃў рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃћрЃЉрЃўрЃА рЃарЃљрЃЮрЃЊрЃћрЃюрЃЮрЃЉрЃљ рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџ рЃцрЃўрЃџрЃўрЃљрЃџрЃерЃў
- рЃцрЃўрЃџрЃўрЃљрЃџрЃўрЃА рЃАрЃърЃћрЃфрЃўрЃцрЃўрЃЎрЃБрЃарЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃўрЃќрЃЮрЃџрЃљрЃфрЃўрЃљ

### рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃўрЃА рЃЏрЃўрЃЏрЃЮрЃ«рЃўрЃџрЃЋрЃљ
- рЃфрЃЮрЃфрЃ«рЃљрЃџрЃў рЃАрЃбрЃљрЃбрЃБрЃАрЃўрЃА рЃўрЃюрЃЊрЃўрЃЎрЃљрЃбрЃЮрЃарЃћрЃЉрЃў (рЃЮрЃюрЃџрЃљрЃўрЃю/рЃЮрЃцрЃџрЃљрЃўрЃю)
- рЃарЃћрЃАрЃБрЃарЃАрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃћрЃЉрЃў
- рЃцрЃљрЃЋрЃЮрЃарЃўрЃбрЃў рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃћрЃЉрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ
- рЃњрЃљрЃцрЃљрЃарЃЌрЃЮрЃћрЃЉрЃБрЃџрЃў рЃФрЃўрЃћрЃЉрЃљ рЃЊрЃљ рЃцрЃўрЃџрЃбрЃарЃљрЃфрЃўрЃљ
- рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃўрЃА рЃ»рЃљрЃюрЃЏрЃарЃЌрЃћрЃџрЃЮрЃЉрЃўрЃА рЃўрЃюрЃЊрЃўрЃЎрЃљрЃбрЃЮрЃарЃћрЃЉрЃў
- рЃАрЃгрЃарЃљрЃцрЃў рЃгрЃЋрЃЊрЃЮрЃЏрЃўрЃА рЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃў

### рЃЊрЃћрЃбрЃљрЃџрЃБрЃарЃў рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ
- рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃДрЃЮрЃЋрЃџрЃўрЃАрЃЏрЃЮрЃЏрЃфрЃЋрЃћрЃџрЃў рЃЏрЃћрЃбрЃарЃўрЃЎрЃћрЃЉрЃў
- AnyDesk-рЃЌрЃљрЃю рЃћрЃарЃЌрЃ»рЃћрЃарЃљрЃЊрЃў рЃЊрЃљрЃгрЃЎрЃљрЃърЃБрЃюрЃћрЃЉрЃўрЃЌ рЃЎрЃљрЃЋрЃерЃўрЃарЃў
- IP рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃў рЃЊрЃљ рЃЦрЃАрЃћрЃџрЃўрЃА рЃЊрЃћрЃбрЃљрЃџрЃћрЃЉрЃў
- рЃўрЃАрЃбрЃЮрЃарЃўрЃБрЃџрЃў рЃгрЃљрЃарЃЏрЃљрЃЊрЃЮрЃЉрЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў
- рЃАрЃљрЃфрЃљрЃЋрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ
- рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃўрЃА рЃЊрЃћрЃбрЃљрЃџрЃћрЃЉрЃў
